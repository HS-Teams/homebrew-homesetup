# HomeSetup
## Terminal .dotfiles and shell imrpovements

### 1. Files

The following files will be added when installing this project:

```
~/.bash_aliases
~/.bash_colors
~/.bash_env
~/.bash_functions
~/.bash_profile
~/.bashrc
```

To override or add (customize) aliases, you create a custom file, as follows:

```
~/.aliases
~/.profile
~/.env
```
