### Profile related functions

### __hhs_activate_nvm

```bash
Usage: __hhs_activate_nvm
```

##### **Purpose**:

Lazy load helper function to initialize NVM for the terminal.

##### **Returns**:

**0** on success; **non-zero** otherwise.

##### **Parameters**: -

##### **Examples:**

```bash
  $ __hhs_activate_nvm
```
